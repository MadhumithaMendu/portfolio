Online Food Ordering System
Project Description

The online food ordering website provides a platform for users to search and discover new restaurants. It offers features such as placing food orders, tracking the status of orders, browsing restaurant menus, viewing customer reviews, and much more.

Features

  • User registration and login
  • Search and filter restaurants
  • View restaurant details and menu
  • Place food orders
  • Track the status of orders
  • Rate and review restaurants
  • Payment integration

Technologies Used

  • Html
  • Css
  • javascript
  • bootstrap
  • php
  • MySQL
